// Define restrições em equipamentos, itens e cartas em tipos de mapas e zonas personalizadas.
//
// Estrutura:
// IDdoItem,Flag
//
// Legenda para o campo 'Flag' (bitmask):
// 1    - restringido em mapas normais
// 2    - restringido em PVP
// 4    - restringido em GVG
// 8    - restringido em BG
// Zonas de restrição - habilitáveis com o mapflag 'restricted <número>'
// 32   - restringido na zona 1
// 64   - restringido na zona 2
// 128  - restringido na zona 3
// 256  - restringido na zona 4
// 512  - restringido na zona 5
// 1024 - restringido na zona 6
// 2048 - restringido na zona 7
// 4096 - restringido na zona 8
//
// Usar valores negativos como Flag irão ter o efeito contrário.
//
// Exemplos:
// 1201,1   // Faca não pode ser equipada em mapas normais
// 608,4    // Semente de Yggdrasil não pode ser consumida em GvG e castelos da WoE
// 4174,6   // Carta Deviling não tme efeito em PVP ou GVG, e nem durante Woe
// 501,32   // Poção Vermelha não pode ser consumida em mapas da zona 'restricted zone 1'
// 519,322  // Leite não pode ser consumido em mapas PVP e em mapas da zona 'restricted zone 2' ou 'restricted zone 4' (256+64+2)
// 519,-2   // Remove a inabilidade de consumir (permite consumo) de Leite em mapas PVP.
